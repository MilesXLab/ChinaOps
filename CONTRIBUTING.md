## How to Contribute

### 1. Report a Problem or Missing Information

**Use this template:**

```markdown
**Issue Type:** [Bug | Missing Info | Outdated Info | Wrong Contact]

**SOP Name:** [Which SOP is affected?]

**Problem Description:**
[What's wrong or missing? Be specific with examples]

**Your Solution/Experience:**
[How did you solve it? What should be documented?]

**Evidence/Source:**
[Links, official sources, dates, personal experience details]

**When did you encounter this?**
[Recent date - helps verify if info is current]
```

**Examples of good issues:**
- "The hospital contact number in Hospital Access SOP is outdated (called today, number is wrong)"
- "Missing: How to get a replacement SIM card if yours is lost/stolen"
- "Correction: The train name format for 12306 doesn't include spaces in surnames"

### 2. Submit a New SOP

**Before you start:**
- [ ] Your topic solves a real problem (you've experienced it)
- [ ] The solution is tested and verified
- [ ] It's not covered in existing SOPs
- [ ] You have time to write 300+ lines

**Follow these steps:**

1. **Copy the template:** [TEMPLATE_ENHANCED_SOP.md](TEMPLATE_ENHANCED_SOP.md)

2. **Fill in all 10 sections:**
   - Problem Statement
   - Root Cause Analysis
   - Step-by-step Solution
   - Comparison tables (if applicable)
   - Miles' Tips (practical advice)
   - Recommendations (specific brands/apps/contacts)
   - Emergency Fallback Plans
   - Local Hacks & Money-saving Tips
   - FAQ (5-10 questions)
   - Contacts & Resources

3. **Quality checks:**
   - [ ] Written for English-speaking travelers
   - [ ] Chinese terms in parentheses with pinyin
   - [ ] Real contact numbers/addresses (verified)
   - [ ] Safety warnings included (use ‚ö†Ô∏è emoji)
   - [ ] Multiple solution paths (if Plan A fails, here's Plan B)
   - [ ] Estimated costs in RMB/USD
   - [ ] Tested information (not just theory)

4. **Submit:**
   ```
   Title: [New SOP] - [Topic Name]
   
   Category: [System Setup / Daily Runtime / Emergency / Parenting]
   
   Description:
   - What problem does this solve?
   - Why is it important?
   - Have you tested this solution?
   - Any special challenges for foreign visitors?
   ```

### 3. Update an Existing SOP

**When to update:**
- ‚úÖ Information is outdated or incorrect
- ‚úÖ Contact numbers/addresses have changed
- ‚úÖ New apps/services are now available
- ‚úÖ Better procedures discovered
- ‚ùå Don't just reword without new info

**How to update:**

```markdown
Title: [Update] [SOP Name] - [Specific Change]

What's changing:
- Old info: [Current content]
- New info: [Corrected/updated content]
- Reason: [Why this change? Source? Recent experience?]

Source/Verification:
- [ ] Personally tested/verified
- [ ] Official source (government/hospital)
- [ ] Recent experience (date: ___)
- [ ] Multiple people confirmed this

Add this line to the SOP:
**Updated:** [Month Year] - [Your brief explanation]
```

### What Makes a Good Contribution

**‚ùå Avoid:**
- Unverified opinions ("I think it might work")
- Generic travel advice (use Google for that)
- Outdated information (more than 6 months old)
- Sponsored recommendations or promotions
- Information not applicable to China
- Very short submissions (less than 300 words for new SOPs)

**‚úÖ Aim for:**
- Personal experience ("I tried this and it worked")
- Specific examples ("The hospital at 123 Main St...")
- Real contact info (verified phone numbers, hours)
- Current as of [recent date]
- Multiple solution paths
- Safety warnings and emergency procedures
- Real costs and time estimates

## ‚ùì Frequently Asked Questions

**Q: Can I write about my hometown/city?**
A: Absolutely! ChinaOps needs local knowledge from all over China. Just specify the city and mention if rules might differ elsewhere.

**Q: What if I find conflicting information?**
A: Note both versions with sources. Example: "Some hospitals accept foreign cards, others require UnionPay (verified at Ruijin Hospital Jan 2026)"

**Q: Can I update something I submitted before?**
A: Yes! Add new info with the updated date. No need to resubmit the whole thing.

**Q: How do I suggest a new SOP topic?**
A: Open an issue titled "[Suggestion] [Topic Name]" and describe the problem you want solved.

**Q: Can I translate SOPs to Chinese?**
A: Yes, but do it in Stage 2. For now, add Chinese translations in brackets as we do now.

**Q: What if I'm not sure about something?**
A: Mark it as uncertain: "I'm not 100% sure, but... [info] [when tested] [what happened]"
